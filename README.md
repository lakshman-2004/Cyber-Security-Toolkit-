# 🔐 Cyber Security Toolkit

A collection of useful **Cyber Security Tools, Scripts, and Resources** designed for students, ethical hackers, penetration testers, and cybersecurity enthusiasts.

This repository provides tools and references that help in **learning cybersecurity, performing security testing, and understanding common cyber threats**.

---

## 📌 Features

- Network Security Tools
- Penetration Testing Utilities
- Password Cracking Tools
- Vulnerability Scanning
- Cybersecurity Learning Resources
- Kali Linux Tool References
- Ethical Hacking Utilities

---

## 🛠️ Tools Categories

### 🔎 Reconnaissance Tools
Tools used for information gathering and target analysis.

Examples:
- Nmap
- Recon-ng
- Whois Lookup
- DNS Enumeration

---

### 🌐 Network Security Tools
Tools used to analyze and monitor network traffic.

Examples:
- Wireshark
- Tcpdump
- Netcat
- Scapy

---

### 🔓 Password & Hash Cracking Tools
Tools used for password auditing and cracking.

Examples:
- John the Ripper
- Hashcat
- Hydra

---

### 🧪 Penetration Testing Tools
Used for ethical hacking and security testing.

Examples:
- Metasploit Framework
- Burp Suite
- OWASP ZAP

---

### 📡 Wireless Security Tools
Used for WiFi security testing.

Examples:
- Aircrack-ng
- Wifite
- Kismet
